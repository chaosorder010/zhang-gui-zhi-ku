# Triage labels

Canonical 5 label 全用默认字符串:

| Label | 角色 |
|---|---|
| `needs-triage` | 需维护者评估 |
| `needs-info` | 等报告者补充 |
| `ready-for-agent` | 已就绪，AFK agent 可接 |
| `ready-for-human` | 需人实现 |
| `wontfix` | 不做 |

GitHub 现状: `ready-for-agent` ✅ / `wontfix` ✅ (默认 label). 缺 `needs-triage` / `needs-info` / `ready-for-human` — 由 triage skill 首次运行时按需建。
