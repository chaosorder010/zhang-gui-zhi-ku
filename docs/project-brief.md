# 掌柜智库 — 项目总纲

> 生成日期: 2026-07-14
> 仓库: chaosorder010/zhang-gui-zhi-ku
> 状态: MVP 闭环完成, 评估驱动优化阶段

---

## 1. 目标

构建企业级 RAG (检索增强生成) 知识库系统, 让维修技师用自然语言提问, 几秒内从已上传的电子产品手册/维修指南中获取准确技术参数。

**核心原则**: 宁缺勿错 — 知识库无答案时直接告知, 不让 LLM 编造技术参数。

## 2. 目标用户

| 角色 | 需求 |
|---|---|
| 维修技师 | 用中文自然语言提问, 秒级获取准确技术参数 |
| 运维工程师 | `docker compose up` 一键部署, 环境变量注入密钥 |

## 3. 约束

- **语言**: 中文为主, 手册含中英混合技术参数
- **部署**: 单机 Docker Compose (FastAPI + Milvus + 前端 nginx)
- **模型**: 本地 BGE-M3 embedding + 云端 LLM (longcat/OpenAI 兼容)
- **开发**: TDD (单元 ms 级 + 集成 TestClient + E2E 手动)
- **网络**: Docker Hub 需 VPN, HuggingFace 需离线模式
